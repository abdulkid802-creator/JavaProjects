## 🕹️ RPG Battle Game

A fully featured text-based, turn-based RPG built from scratch in Java.

**Features:**
- Turn-based combat with Attack, Use Items, Special Ability, and Run options
- Critical hits, miss mechanic, and enemy counter-attacks
- Power Strike special ability with a 2 turn cooldown
- Boss battles every 5 rounds with unique special moves
- Inventory system with 4 item types and random enemy drops
- XP and levelling system with scaling stats
- 10 enemies that grow stronger every round

**Concepts used:** Abstract classes, inheritance, encapsulation, ArrayLists, Scanner, Random, switch expressions

## 🕹️ Featured Project — Java RPG Battle Game

A text-based, turn-based RPG built entirely from scratch in Java.

**What's in it:**
- Turn-based combat with a player choice menu
- Critical hits, miss mechanic, and enemy counter-attacks
- Power Strike special ability with a cooldown system
- Boss battles every 5 rounds with unique special moves
- Inventory system with 4 item types and random enemy drops
- XP and levelling system with scaling stats
- 10 enemies that grow stronger every round

**Built with:** Java, OOP (abstract classes, inheritance, encapsulation), ArrayLists, Scanner, Random

🔗 [View the project](https://github.com/abdulkid802-creator/JavaProjects/tree/main/RPGGame)
